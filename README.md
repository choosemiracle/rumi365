# 365 Days with Rumi

一个中英文每日鲁米灵修学习网站，用于个人内在探索、默观练习和日记记录。

## 功能

- 从附件 PDF 抽取每日英文条目，并按原书月份/星期顺序展示。
- 按日期自动映射到学习条目，也可以手动选择条目编号。
- 中英双语、仅中文、仅英文三种阅读模式。
- 每日诗性默想、内在提问、灵修练习。
- 本地浏览器日记保存，不上传个人内容。
- 学习路径视图，标记已经写过日记的条目。
- 支持打印当天学习页。

## 内容说明

本项目从附件 `365 Days with RUMI` PDF 中抽取英文条目，供个人学习使用。当前 PDF 文本层实际可抽取 335 个每日条目和 12 个月份题词，并非完整 365 个自然日条目。七月缺一个星期六条目，脚本按 PDF 可读文本原样保留。

公开部署前请确认你拥有重发布原书摘录及中文译文的授权。

## 翻译

`data/rumi-days.json` 支持 `quoteZh` 字段。可运行：

```bash
/usr/bin/python3 scripts/translate_rumi_days.py
```

该脚本会调用 `OPENAI_API_KEY` 对英文条目分批生成中文译写。若网络不可达，可手动补入或改用其他翻译服务。

## 本地使用

由于页面需要读取 `data/rumi-days.json`，请通过本地服务器访问：

```bash
python3 -m http.server 4187
```

然后打开 `http://localhost:4187/`。

仓库发布到 GitHub 后，在 Settings → Pages 中选择当前分支和根目录即可部署。
